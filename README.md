# Gdscript Library - Useful Tools Script
This repository contains a Gdscript library providing various life-saving tools and functions for Game Dev.

## Installation
From Github:
copy the `addons/gdLib` folder to your Godot project's `addons` folder.

## Features
- **function library**: provides a set of utility functions.
- **unit tests**: provides a set of tests for modules.
